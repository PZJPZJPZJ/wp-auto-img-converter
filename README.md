# WP Auto Img Converter

WP Auto Img Converter 是一款为 WordPress 开发的自动化图片转换插件。它能在图片上传过程中自动将其转换为 WebP 格式并进行压缩，从而提升网站加载速度和性能。

## 🚀 功能特性

- ✅ **自动转换**：上传图片（JPG, PNG, GIF）时自动转换为高压缩率的 WebP 格式。
- ✅ **批量处理**：支持扫描媒体库中现有的非 WebP 图片，并按文件夹分类显示。
- ✅ **按需转换**：在批量转换界面，用户可以勾选特定的文件夹或单张图片进行转换。
- ✅ **层级进度显示**：批量转换时不仅有全局进度条，各目录层级也会显示剩余/总数的实时进度。
- ✅ **原始大图支持**：完整处理 WordPress 的 `-scaled` 机制，确保原始无损图也被正确转换。
- ✅ **质量可控**：支持自定义 WebP 图片质量（1-100）。
- ✅ **灵活的文件策略**：可选择转换后是否保留原始图片文件。

## 🔧 安装使用

### 1. 安装
1. 将插件文件夹上传到 `/wp-content/plugins/` 目录。
2. 在 WordPress 后台“插件”菜单中激活 **WP Auto Img Converter**。

### 2. 配置
- 进入 `设置 > ImgConverter`。
- **启用自动转换**：勾选后，新上传的图片将自动处理。
- **转换格式**：选择需要转换的原始格式（JPG/PNG/GIF）。
- **图片质量**：设置 WebP 质量，建议 75-85。
- **保留原图**：如果希望节省服务器空间，可取消勾选。

### 3. 批量处理现有图片
1. 在设置页面点击“开始扫描”。
2. 插件将扫描所有非 WebP 格式的图片，并按存储路径进行分组。
3. 勾选你想要转换的目录或图片。
4. 点击“批量转换为 WebP”，等待转换完成。

## 🔍 系统要求

- PHP 7.4+
- WordPress 6.7+
- 服务器需安装 GD 库或 Imagick 扩展以支持 WebP 转换。
